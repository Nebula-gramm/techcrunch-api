API Documentation

A brief overview of the TechCrunch Unofficial API and how to use it. This API allows you to fetch articles and data from TechCrunch programmatically. All endpoints return data in JSON format.

GET /latest

Fetches the latest articles from TechCrunch. This is the primary endpoint for getting the most recent content.

{ "id": "2595304", "title": "Apple is bringing RCS to the iPhone", "thumbnail": "https://...", "link": "https://techcrunch.com/...", "description": "It’s happening. Apple...", "author": "Ivan Mehta", "date": "2024-11-16T15:00:00Z" }

GET /c/:category

Fetches articles from a specific category, such as 'startups' or 'ai'. Replace `:category` with the desired category slug.

{ "id": "2595112", "title": "How AI is reshaping venture capital", "thumbnail": "https://...", "link": "https://techcrunch.com/...", "description": "Venture capitalists are...", "author": "Kyle Wiggers", "date": "2024-11-15T18:30:00Z" }

GET /t/:tag

Fetches articles associated with a specific tag. Replace `:tag` with the desired tag slug.

GET /p/:page

Paginates through all articles. Replace `:page` with the page number you want to retrieve.

GET /p/:page/a/:id

Fetches a specific article by its ID from a specific page. This is useful for deep linking into paginated results.

GET /:id

Fetches a single, specific article by its unique ID. Replace `:id` with the article's ID.
